# lanyard-profile-readme

🎟️ Utilize [Cardzera](https://github.com/matheusaudibert/cardzera) to display your Discord server invite card in your GitHub Profile

## Usage

First, [invite the Cardzera bot](https://discord.com/oauth2/authorize?client_id=1368283106276147200&permissions=525312&integration_type=0&scope=bot) to your server (if you haven't already) for this to work.

In a `README.md` file, include the following, replacing `:id` with your Discord server ID:

```md
[![Discord Server Card](https://cardzera.audibert.dev/api/:serverId?t={timestamp})](https://discord.gg/your_invite)
```

It should display something similar to the following (I am using my Discord server ID as an example):

[![Discord Server Card](https://cardzera.audibert.dev/api/1112920281367973900?t={timestamp})](https://discord.gg/programador)

When others click it, they will be directed to your Discord server. Neat!
